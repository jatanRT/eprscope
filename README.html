eprscope

Overview

The key objective of this fully open source package is to bring handy functions corresponding to “everyday” data processing/analysis in the EPR (Electron Paramagnetic Resonance) spectroscopy mainly in chemistry. The package doesn’t want to replace the excellent EPR simulation/processing EasySpin Toolbox for MATLAB . Rather, it may be considered like a complimentary package/toolbox with practical functions which have to be otherwise performed by several tools, e.g. like by the acquisition/processing software, supplied by the EPR spectrometer manufacturers (see e.g. Xenon/WinEPR) as well as by several other proprietary software platforms like MS Office and/or Orgin/SigmaPlot/Igor. Therefore, the {eprscope} also tries to reduce such many steps if the above-mentioned software combination would be adopted.

Installation

## So far, the package can be installed by =>

# install.packages("devtools")
# devtools::install_github("jatanRT/eprscope.git")

Before the installation, please make sure that you have already followed instructions for the {nloptr} package installation depending on your operating system. This package is required for the proper running of optimization/fitting functions of the {eprscope} .

Completely new users or people who haven’t already installed the R environment, please consult ➨

  1. the R installation procedure

  2. installation of the Rstudio IDE

  3. the latest R tools release ONLY for WINDOWS OS

Additionally, an open-source scientific and technical publishing system Quarto together with Pandoc, a document converter system, may be needed for sharing the results coming from {eprscope} in desired formats like pdf , html , docx , pptx or tex.
